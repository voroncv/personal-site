## Personal site
### Personal site on Vue.js, Webpack, Firebase Semantic UI, Express.js and Node.js
## Clone rep
```
$ git clone https://github.com/voroncv/personal-site.git
$ cd personal-site
```
## Install and setup
```
$ npm install
$ npm run dev
```
## Build project
```
$ npm run build
```
